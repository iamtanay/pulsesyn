| `bias_test.go` | Full test suite — see count below. |

**Bias coefficient formula implemented:**
```
mean_deviation   = mean( |score(validator_verdict) - population_average_score| )
bias_coefficient = mean_deviation / 1.0  (max_possible_deviation = 1.0)
```

Verdict score encoding: SUPPORTED=1.0, UNSUPPORTED=0.0, MISLEADING=0.5, INDETERMINATE=0.5.

## 4. Key Design Decisions Made

See `DECISIONS.md` for the full entries. Summary:

| Decision | Rationale |
|---|---|
| Bias uses linear verdict score axis | Minimal faithful encoding; multi-axis deferred until empirical data justifies it |
| Window uses FIFO eviction | Direct implementation of spec "last N validations" language |
| testify formalised in go.mod | Was implicitly required by Session 1 tests; now explicit |
| Simulation uses uniform random selection | Composite score selector is Phase 3 work; uniform sufficient for Phase 1 math validation |
| `injectDomainReputation` uses public API only | Preserves immutability invariant; simulation is production code, not a test |
